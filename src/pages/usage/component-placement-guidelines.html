Component placement guidelines

Prerequisites

Before proceedings with the USPTO Design System, we assume that you are already aware of the basics of HTML, CSS, and Twitter Bootstrap. If you are not aware, then we suggest that you go through an overview of the Bootstrap documentation.

For the development of the user interface, we assume that you are familiar with the basic design requirements of web components, and their respective accessibility guidelines, within the context and display area of an application.

Usage

Refer to the USPTO-DS Component Placement Guidelines for the preferred placements of each common component or pattern that you plan to use in your design solution.

For more information regarding components, visit the “Components” section of the USPTO-DS.
For more information regarding patterns, visit the “Patterns” section of the USPTO-DS.
