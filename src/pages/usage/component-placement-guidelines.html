Component placement guide

Prerequisites

Before proceedings with the USPTO Design System, we assume that you are already aware of the basics of HTML, CSS, and Twitter Bootstrap. If you are not aware, then we suggest that you go through an overview of the Bootstrap documentation.

For the development of user interfaces, familiarize yourself with basic design requirements of web components and their respective accessibility guidelines.

Usage

Refer to the USPTO-DS Component Placement Guide for the preferred placements of each common component and/or pattern that you plan to use in your design solution.

For more information regarding components, visit the Components section of the USPTO-DS.
For more information regarding patterns, visit the Patterns section of the USPTO-DS.

Additional resources

  • U.S. Web Design System (USWDS)
