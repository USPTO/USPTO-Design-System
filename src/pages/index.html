USPTO Design System

An open source Design System, UI style guide, pattern library, and Bootstrap theme.

Explore the Design System Download
Current version: 2.0.4-Beta

What is this?

The USPTO Design System is an open source UI style guide, pattern library, and Bootstrap theme. Our docs are packed full of usability guidelines, examples, and source code. This design system is based on the Bootstrap, utilizing components and design elements from the USWDS.

This design system is a continious work in progress, so expect regular changes and monthly releases.

How can I use it?

Once downloaded from the link above, you can use either the minified CSS build or the source SCSS files in your own build system. View our how-to guide

Who is it for?

The USPTO Design System is intended for designers and developers working on USPTO related projects. But, since it's open source, anyone can take advantage of it. If it's useful to you, use it.

Why did we make it?

We have a lot of web projects at the USPTO. These design guidelines and resources aim to create a unified experience across all projects at the USPTO, both internal and external.

Not using Bootstrap?

That's fine! A lot of our projects use Bootstrap, but not all of them. The usability guidelines and other resources could still be useful.

Want to contribute?

We want your help! Visit our Github repository for contribution guidelines. If you have improvements or bug fixes, let us know!
