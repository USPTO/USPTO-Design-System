Browser support

Modern web browsers

USPTO Design System components are built on a solid HTML foundation, progressively enhanced to provide core experiences across browsers. All users will have access to the same critical information and basic experience regardless of their browser, although those experiences will render significantly better in newer browsers.

The USPTO Design System supports the latest, stable releases of all major browsers and platforms. Learn more about web browser standards.

Mobile devices and browsers

Generally speaking, the USPTO Design System supports the latest versions of each major platformâ€™s default browsers. Mobile browser support will greatly depend on the manner in which your UI was implemented. Because this library is built on top of Boostrap, all mobile devices supported by Boostrap will render accurately.

Internet Explorer

Internet Explorer 10+ is supported, including Edge; IE9 and down is not. Please be aware that some CSS3 properties and HTML5 elements are not fully supported in IE10, or require prefixed properties for full functionality. If your application requires or targets legacy browser compatibility, we recommend using our older USPTO Design Pattern Library.
